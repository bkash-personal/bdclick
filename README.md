# BDSocialPro - SMM Panel Website

বাংলাদেশের সেরা SMM (Social Media Marketing) Panel ওয়েবসাইট টেমপ্লেট। এই প্রোজেক্টটি bdclick24.com এর মতো একটি SMM Panel ওয়েবসাইট তৈরি করার জন্য তৈরি করা হয়েছে।

## ✨ ফিচার সমূহ

- 🎨 আকর্ষণীয় ও আধুনিক UI ডিজাইন
- 📱 সম্পূর্ণ রেসপন্সিভ (Mobile, Tablet, Desktop)
- 🇧🇩 বাংলা ও ইংরেজি ভাষা সাপোর্ট
- 💳 বিকাশ, নগদ, রকেট, PayPal পেমেন্ট UI
- 🔒 লগইন ও রেজিস্ট্রেশন ফর্ম
- 📊 সার্ভিস কার্ড (Facebook, Instagram, YouTube, TikTok, Twitter, Telegram)
- 💰 প্রাইসিং সেকশন
- 📞 কন্টাক্ট ফর্ম
- ⚡ Fast Loading & Optimized

## 🚀 GitHub Pages এ হোস্ট করুন

### সহজ পদ্ধতি (Recommended):

1. **এই রিপোজিটরি ফর্ক করুন**
   - এই রিপো GitHub এ ফর্ক করুন

2. **Settings > Pages এ যান**
   - Source হিসেবে `main` branch এবং `/ (root)` সিলেক্ট করুন

3. **একটু অপেক্ষা করুন**
   - GitHub আপনার সাইট পাবলিশ করবে
   - URL হবে: `https://yourusername.github.io/smm-panel-website/`

### কাস্টম ডোমেইন ব্যবহার করতে:
1. Settings > Pages > Custom domain এ আপনার ডোমেইন যোগ করুন
2. DNS এ GitHub IP পয়েন্ট করুন
